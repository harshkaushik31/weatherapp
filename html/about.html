About the Weather app

This website tells the weather data of various cities including cities in India and other contries also. It not only tells the weather but also tells various other parameter which are listed below.

  • • Cloud PCT - This is the cloud cover percentage over a city. Basically it tells the percentage of area of city which is covered in cloud.
    Higher this number the cloudier the day is and lower the number the less cloudier the weather is.
  • • Humidity - This parameter tell the humidity present in the air. It is the percentage of water molecules present in the air.
    The higher the humidity percentage the weather will be more humid and vice-versa.
  • • Temprature - This tells the current Temprature of the day.
  • • Max Temprature - This tells the maximum Temprature which is expected to reach within the day.
  • • Min Temprature - This tells the minimum Temprature which is expected to reach within the day.
  • • Sunrise Time - This tells the time of sunrise in IST (Indian Standard Time.)
  • • Sunset Time - This tells the time of sunset in IST (Indian Standard Time.)
  • • Wind Degrees - This tells the degree with respect to North from which the wind is blowing.
  • • Wind Speed - This tells the speed with which the wind is blowing.

This website is made using HTML5, CSS3 and Vanilla JavaScript.This website uses Weather API by API Ninjas from Rapid API to fetch the weather data in JavaScript and then it displays the data in the website.
This website also has a search functionality with which you can search the weather for different cities.
